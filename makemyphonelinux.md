# 안드로이드에서 리눅스 설치하기 (루팅 불필요)

Termux 앱을 이용해서 안드로이드 폰/태블릿에서 리눅스 데스크톱 환경을 돌리는 방법. 루팅 없이 가능하다는 게 핵심.

---

## 설치 순서

### 1단계: 저장소 접근 권한 부여

Termux 앱을 열고 아래 명령어를 입력. Termux가 안드로이드의 내부 저장소(사진, 다운로드 폴더 등)에 접근할 수 있게 해준다.

```bash
termux-setup-storage
```

> **참고:** 원문에 `termox`라고 오타가 있는데, 정확한 명령어는 `termux-setup-storage`이다.

### 2단계: 패키지 업데이트

Termux에 설치된 기본 패키지들을 최신 버전으로 업데이트. 도중에 Y/N 물어보면 Y 선택.

```bash
pkg update && pkg upgrade -y
```

### 3단계: Git 설치

설치 스크립트를 GitHub에서 다운받기 위해 Git을 설치.

```bash
pkg install git -y
```

### 4단계: 설치 저장소 클론

GitHub에서 리눅스 데스크톱 설치 스크립트를 다운로드.

```bash
git clone https://github.com/lucmsilva651/termux-desktop
```

### 5단계: 다운로드된 디렉토리로 이동

```bash
cd termux-desktop
```

### 6단계: 설치 스크립트에 실행 권한 부여

```bash
chmod +x setup.sh
```

### 7단계: 설치 실행

이 단계에서 데스크톱 환경 선택 등 실제 설치가 진행된다. 시간이 좀 걸릴 수 있음.

```bash
./setup.sh
```

### 8단계: 리눅스 데스크톱 시작

```bash
./start-desktop
```

## 데스크톱 환경 선택 시 주의사항

설치 중에 어떤 데스크톱 환경을 쓸지 선택하는 단계가 나온다. 자기 폰의 AP(프로세서)에 따라 다르게 선택해야 한다.

| 칩셋 | 추천 옵션 | 이유 |
|------|-----------|------|
| **스냅드래곤 (Qualcomm)** | XFCE + Turnip/Zink | 하드웨어 가속(GPU 활용)이 되어 훨씬 부드럽고 빠름 |
| **미디어텍 (MediaTek)** | LXQT 또는 XFCE (소프트웨어 렌더링) | Turnip/Zink 드라이버 미지원, 하드웨어 가속 시 불안정 |
| **엑시노스 (Exynos)** | LXQT 또는 XFCE (소프트웨어 렌더링) | Turnip/Zink 드라이버 미지원, 하드웨어 가속 시 불안정 |

## "팬텀 프로세스" 문제 해결

안드로이드 12 이상에서 리눅스 세션이 갑자기 죽거나 크래시가 나는 경우가 있다. 안드로이드가 백그라운드에서 돌아가는 자식 프로세스를 강제로 죽이기 때문.

### 해결법

1. **개발자 옵션 활성화:** 설정 → 휴대전화 정보 → 빌드 번호 7번 탭
2. **설정 → 개발자 옵션** 에서 **"자식 프로세스 제한 비활성화"** (Child process restrictions / Phantom process killer) 항목을 꺼준다.

## 매일 사용할 때 (설치 완료 후)

한 번 설치가 끝나면, 리눅스를 쓸 때마다 아래 3단계만 반복하면 된다.

| 순서 | 동작 |
|------|------|
| 1 | **Termux-X11 앱**을 먼저 열어서 백그라운드에 놔둠 (화면 출력 역할) |
| 2 | **Termux 앱**을 열고 `./start-desktop` 입력 |
| 3 | **Termux-X11 앱**으로 전환하면 리눅스 데스크톱이 보임 |

### 구조 설명

- **Termux** = 터미널 (명령어 창)
- **Termux-X11** = X11 디스플레이 서버 역할, 리눅스 데스크톱의 "모니터" 같은 것
